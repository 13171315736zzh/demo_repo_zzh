# 俄罗斯方块 (Tetris)

Python + Pygame 实现的俄罗斯方块小游戏。

## 运行方式

1. 激活虚拟环境：
   ```bash
   source nenv/bin/activate
   ```

2. 运行游戏：
   ```bash
   python tetris.py
   ```

## 操作说明

- **← / →**：左右移动
- **↑**：旋转方块
- **↓**：加速下落
- **空格**：硬降落（直接落到底部）
- **R**：游戏结束后重新开始

## 依赖

- Python 3.x
- pygame >= 2.5.0（由 nenv 虚拟环境管理）
